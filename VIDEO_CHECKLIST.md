# Video Demo Checklist

## Pre-Recording Setup
- [ ] Clear browser cache
- [ ] Set browser zoom to 100%
- [ ] Close unnecessary tabs
- [ ] Test microphone
- [ ] Practice run-through
- [ ] Open https://cs-webapps.bu.edu/xiea/

## Recording Order (3 minutes)

### 1. Introduction (15 sec)
- [ ] State your name and project
- [ ] Brief overview of features

### 2. Home Page (20 sec)
- [ ] Show statistics dashboard
- [ ] Point out navigation
- [ ] Show featured content

### 3. Destinations (30 sec)
- [ ] Click Destinations
- [ ] Browse list
- [ ] Open Paris detail page
- [ ] Show related packages

### 4. Packages (40 sec)
- [ ] Click Packages
- [ ] Demonstrate search box
- [ ] Use destination filter
- [ ] Use price filter
- [ ] Click package to show details
- [ ] Show full itinerary

### 5. Create Booking (25 sec)
- [ ] Click "Create New Booking"
- [ ] Select customer
- [ ] Select package
- [ ] Enter number of people
- [ ] Show auto-calculated price
- [ ] Submit form

### 6. View & Update (20 sec)
- [ ] Go to Bookings list
- [ ] Use status filter
- [ ] Open booking detail
- [ ] Click Edit
- [ ] Change status to "Confirmed"
- [ ] Save

### 7. Delete Demo (15 sec)
- [ ] Navigate to any record
- [ ] Click Delete
- [ ] Show confirmation warning
- [ ] Click Cancel (don't delete)

### 8. Relationships (15 sec)
- [ ] Show destination → packages link
- [ ] Show customer → bookings
- [ ] Show booking → package → destination chain

### 9. Conclusion (10 sec)
- [ ] Summarize features demonstrated
- [ ] Thank the viewer

## Post-Recording
- [ ] Review video
- [ ] Upload to YouTube (Unlisted)
- [ ] Copy URL
- [ ] Update video_url.txt
- [ ] Commit and push to GitHub

## Recording Tools

### Windows:
- Xbox Game Bar: Windows + G
- OBS Studio: https://obsproject.com/

### Settings:
- Resolution: 1920x1080 or 1280x720
- FPS: 30
- Enable microphone

## Tips
- Speak clearly and at moderate pace
- Don't rush - 3 minutes is plenty of time
- If you make a mistake, pause and continue
- You can edit multiple takes together
- Show the features, not the code
- Be enthusiastic!

## What NOT to Show
- ❌ Source code
- ❌ Database admin interface
- ❌ Django admin (unless briefly at end)
- ❌ Command line/terminal
- ❌ File structure

## What TO Show
- ✅ User-facing web interface only
- ✅ Smooth navigation
- ✅ All CRUD operations
- ✅ Search and filtering
- ✅ Professional UI
- ✅ Data relationships
